# SmartTAg parSER


## Install
From PyPI:

```bash
pip install -r requirements.txt
```
Or
```bash
pip install --no-cache-dir --upgrade -r requirements.txt
```

## Run
```bash
streamlit run app.py
```
Or
```bash
python -m streamlit run app.py
```
